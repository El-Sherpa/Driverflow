# 📘 Manual de Usuario - DriveFlow Mobile

¡Bienvenido a DriveFlow! Esta guía te ayudará a navegar y utilizar todas las funciones de tu nueva plataforma de gestión de flotas.

## 1. Inicio de Sesión
- Ingresa tus credenciales proporcionadas.
- Si eres un usuario nuevo, puedes registrarte o ingresar el código de empresa si tu organización ya está registrada.

## 2. Dashboard Principal
Dependiendo de tu rol (Admin, Empresa, Agente o Cliente), verás un panel personalizado:
- **Admin/Empresa**: Resumen de usuarios activos, vehículos y estadísticas globales.
- **Cliente**: Estado de su vehículo asignado y accesos rápidos.

## 3. Uso del GPS 🗺️
Para acceder al rastreo:
1. Ve a la sección **GPS** en el menú lateral.
2. **Encender GPS**: Pulsa el botón "ENCENDER" para activar el rastreo. Verás el badge "GPS ONLINE" cuando esté activo.
3. **Buscar Destino**: Utiliza la barra superior para ingresar una dirección.
4. **Trazar Ruta**: Pulsa el botón de cuadrícula (FAB inferior) para calcular la mejor ruta hacia el destino. Verás el tiempo estimado (ETA) y la distancia en el panel inferior.
5. **Apagar GPS**: Pulsa "APAGAR" para detener el rastreo y limpiar el mapa.

## 4. Gestión de Perfil
- En "Mi Perfil" puedes actualizar tus datos personales y cambiar tu contraseña.

## 5. Ajustes y Personalización
- **Idioma**: Puedes cambiar entre Español e Inglés desde el menú de Ajustes.
- **Notificaciones**: Configura las alertas que deseas recibir.

## 6. Soporte Técnico
Si tienes problemas con la aplicación, contacta al administrador de tu flota o envía un correo a soporte@driveflow.com.
